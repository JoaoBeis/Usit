Usit, web app final sem frameworks

Como abrir
1. Extrair o zip.
2. Abrir index.html no Chrome, Edge ou Firefox.
3. Para teste mais estável, abrir a pasta no VS Code e usar Live Server.

Nota sobre acesso reservado
Não existe botão público para administrador ou parceiro. O acesso é feito pelo mesmo formulário de login e a área apresentada depende da função associada às credenciais.

Dados locais
A aplicação guarda dados em localStorage. Anúncios, reservas, mensagens e alterações ao perfil ficam guardados no browser.

Para limpar dados de teste
Abrir a consola do browser e executar:
localStorage.removeItem('usit_final_plain_v1')
Depois recarregar a página.
